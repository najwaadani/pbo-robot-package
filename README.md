# Robot Package System

This project is a simple Java-based application developed to simulate a robot package system. It demonstrates object-oriented programming (OOP) concepts such as classes, objects, and basic logic implementation.

## Features
- Simulates robot operations in handling packages
- Demonstrates object-oriented programming concepts
- Simple and easy-to-understand program structure

## Technologies
- Java

## Description
This project was developed as part of academic coursework and is intended to demonstrate basic programming logic and object-oriented programming principles.

## How to Run
1. Open the project in your Java IDE (e.g., NetBeans, IntelliJ, Eclipse)
2. Compile and run the main class
3. Observe the output in the console
